# Mike Portfolio - Senior ASP.NET Developer

A professional portfolio website built with .NET 8 Razor Pages, showcasing projects and skills.

## Features
- Modern responsive design with Bootstrap 5
- Project showcase with case studies
- Professional timeline of experience
- CV download functionality
- Deployable to Render with Docker

## Local Development

1. **Prerequisites**
   - .NET 8 SDK
   - Git

2. **Run Locally**
   ```bash
   git clone https://github.com/yourusername/mike-portfolio.git
   cd mike-portfolio
   dotnet restore
